Padding and Margin

  •     Padding is for creating space between an element and a border that surrounds the element while margin is for creating space outside the border. 

Block and inline elements

  •     block element starts with a new line and takes the full length of width. For example, we can specify the CSS of an element to be "display: block;" to take the entire width and start in a new line.
  •     inline elements don't take more space than the width of the element and don't start in a new line      

Position: for positioning an element. There are 4 types, static, fixed, absolute, relative. The default is static and relative is the same as static unless we specify top, left, right, bottom properties. 
Floats and Clear: float is for positioning an element to left or right and clear is to specify that an element to move below a floating element.  we must also specify the width of a floating element not to take up the entire width.
